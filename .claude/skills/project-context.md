# Project Context Skill

## 프로젝트 개요
**배분보충-AIA**: F&F S-ERP 유통ERP 시스템의 AI 기반 보충배분 화면

## 비즈니스 플로우

### 데이터 전제조건
- SC-Total / SC-Shop으로 앙상블하여 매장별 판매 예측값은 DB에 이미 존재

### 조회 조건 (전부 필수값)
| 필드 | 설명 |
|------|------|
| 브랜드 | M(MLB), X(Discovery), V(Duvetica), ST(Sergio Tacchini), I(MLB KIDS) |
| 시즌 | 운영시즌 (예: 26S, 25F) |
| AP | 배분 유형 (오프라인 일반, 온라인 등) |
| 스타일 | 스타일코드 |
| 컬러 | 컬러코드 |

### 화면 플로우
1. 유저가 보충배분 원하는 스타일-컬러 입력
2. "조회" 버튼 클릭
3. 해당 스타일-컬러의 사이즈별 물류재고, 배분대상매장의 재고량 API 조회
4. 기입고 매장들에 한하여 추가 보충 필요 수량 표시
5. 화면에서 수량 조정
6. "엑셀 다운로드" 버튼 클릭
7. 엑셀 템플릿에 "AP → 매장" 형식으로 다운로드

### 추후 개발 예정
- [ ] 보충배분 추천되는 스타일-컬러 AI 추천
- [ ] 입고 진행되지 않았던 매장도 추가 가능
- [ ] 매장간 RT 진행 수량도 표시
